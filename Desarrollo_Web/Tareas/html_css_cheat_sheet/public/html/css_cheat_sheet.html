HTML Cheat Sheet

Border

Border define el borde que hay alrededor de un elemento

Border-left, right, top, bottom

Border-left define el borde que hay a la izquierda de un elemento

Border-right define el borde que hay a la derecha de un elemento

Border-top define el borde que hay arriba de un elemento

Border-bottom define el borde que hay debajo de un elemento

Border-radius

Border-radius define el radio que va a tener el borde, esto genera que el borde tenga las esquinas curveadas

Background-image

Background-image define la imagen de fondo que tendrá el elemento

Color

Color define el color del texto del elemento

Font

Permite establecer de una sola vez los valores para varias propiedades relacionadas con la tipografía.

Font-family

Font-family define el tipo de fuente que se va a usar

Font-style,weight

Font-style define el aspecto de una familia tipografica

Font-weight define el grosor de la letra

Width, height

Width define el ancho de un elemento

Height define el alto de un elemento

List-style

List-style permite personalizar los marcadores de los elementos de lista

    Padding

    Padding permite agregar espacio en el interior de un elemento, entre su contenido y su borde

    Overflow

    Overflow controla cómo se maneja el contenido cuando excede los límites de su contenedor
